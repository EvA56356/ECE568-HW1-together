Last updated: 2024-02-02

Driver Intro

If you have a car, you can register as a driver. Drivers can search for open ride requests, claim and start a ride service. A user can only be registered as a driver if the user is logged in. A registered driver should be able to search for open ride requests (filtered by the driver's vehicle capacity and type / special info, if applicable). Driver should have car (obviously) if they want to register as a driver. A logged in User should be able to register as a driver by entering their personal & vehicle info.

How to register as a driver?

To begin with, login as a normal duber user. After that, go to setting - driver info - click "edit"

Choose 'Yes' for "Are you a driver", and set up your driver info, including:
    1) Vehicle Type
    2) License Plate Number
    3) Maximum Number of Passenger
    4) Special Info (optional)

Once you click "Save", you are now ready as a driver!
