Duber Owner User Guide

Last updated: 2024-02-02

Owner Intro

Most of the duber users are owner when they signed up. Owner is a concept when it comes to the ride request. Owner is the person who creates the ride request.

How to register as a owner?

There is no specific step to become an "owner" (owner is a concept when it comes to the ride request). Once you signup and login, you are officially an user. You can create a ride request and become an owner.

How to request a ride

Go to 'Request an ride' and fill in the form. Requesting a ride should involve specifying:
   1) a required arrival (date & time)
   2) the number of total passengers from the owner’s party
   3) [optional] a vehicle type and any other special requests info. After submitting the form, the ride request will be created and the owner will be able to see the ride request in the 'My Rides' page.

How to edit a ride

Go to any of the ride at "My Rides" section and select "Edit" button.

Edit anything you like as long as the ride status is OPEN (Note: You cannot edit the ride once the ride status changed to CONFIRMED)

There are two scenarios that worth mentioning:
   1) You can edit the number of passenger if the ride has not been picked by a driver. However, if you have specified the desired vehicle, then the number of passenger cannot exceed the maximum number of vehicle capacity.
   2) If one driver has been assigned with this ride, you cannot edit anything else except the number of passenger in your side. Also, you cannot have passenger more than the vehicle capacity.

If the user designated passenger number exceeds the number of passengers the vehicle can contain, an error will be given. After editing, click "Save" button to save the changes.

How to cancel a ride

While it is not encouraged to cancel the ride, our policy stipulates that as long as the ride's status is OPEN, the owner can choose to cancel the ride. After cancel the ride, the driver and sharer will receive a email notification states that the ride has been cancelled. The ride cannot be cancelled after the driver has picked the user and confirm the ride (which means the ride's status has been changed to CONFIRM
