Sharer Use Guide

Last updated: 2024-02-02

Sharer Intro

As a sharer, you can use search function to find a required ride to join. 

How to join a ride?

To begin with, click the Seach rides button on the top and select the role, which is 'Sharer'. 

After entering this function, you should enter the following info to search any rides. The system will automatically find the satisfied rides according to the info you provided.
    1) Destination Address
    2) Arrival Time Window
    3) Number of Passenger
By clicking the search button, the page will redirect to the result page, while you can click join button to join a specific ride.

How to edit your info after joining?

If you want to edit your info after join a ride, you can click the Detail button of a ride and click edit button to the following page. Here, you are only allowed to edit you passenger number. The input number should be larger than 0. The system will also detect whether the number is acceptable for this ride according to the car type and current number of passenger.
